**IDEA设置注释模板入口：**

类头注释设置方法：

`打开file->setting->Editor->Filr and Code Templates->Includes->File Header`
 
类头注释模板：
```
/**
 * @Project ${PROJECT_NAME}
 * @Description:
 * @author TongWei.Chen ${YEAR}-${MONTH}-${DAY} ${HOUR}:${MINUTE}:${SECOND}
 */
```

方法注释模板：
```
/** 
* 方法描述
* 
* @param   
* @return 
*/
```
