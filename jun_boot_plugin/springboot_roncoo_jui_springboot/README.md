# 龙果开源 roncoo-jui-springboot
* 该项目是为了大家更好地运用Spring Boot的功能，进行实战。
* 如果没有使用过Spring Boot，也是一个学习的好项目。
* 可以快速实现一个基于Spring Boot的后台管理系统，前端是基于JUI。
