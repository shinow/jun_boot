# springboot-freemarker 介绍 #

<pre>它是一个典型的MVC三层框架，快速简单的上手。</pre>

#### springboot-freemarker

+ 包含框架有：SpringBoot、SpringMVC、MyBaits、Bootstrap3、Druid、Freemarker；
+ 集成示例有：增删改查及分页;防XSS、SQL注入；

#### 示例启动
  
+ 启动工程；
+ 运行cn.springboot.Application当中的main方法；
+ 浏览器访问工程查看示例效果；
